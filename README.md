## Visão geral
Este projeto foi desenvolvido como parte da disciplina de Projeto Orientado a Objetos da UNIFESP, com o objetivo de aplicar conceitos de padrões de projeto. O projeto consiste em desenvolver um sistema de reservas de salas de estudo para um campus universitátio. Essa aplicação deve permitir que estudantes e professores consultem a disponibilidade de salas,
façam reservas e recebam notificações sobre alterações ou conflitos.
